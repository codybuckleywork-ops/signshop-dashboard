Signshop Dashboard

A production system for sign shops that runs on your own PC. Track every job from design to install, manage inventory, plan around the weather, and get a daily overview in your inbox.

View on GitHub Quick start

Built around a real shop pipeline

Every job moves through the same stages your shop floor does — advance it with a click or drag it on the board.

Design→ Print→ Laminate→ Cut→ Mask→ Install→ Complete

Everything in one place

📋

Job tracking

List, kanban board, week planner, and month calendar views. Customer-first cards, work-order printing, checklists, files & proof approvals.

🌦️

Weather-aware scheduling

7-day forecast with install jobs pinned to their day, live radar, and warnings with real numbers — "rain likely (70%)" on install day, at the install location.

📦

Inventory management

Product codes, vendors, costs, reorder points with low-stock alerts, auto-deducting material usage, and paste-a-URL product autofill.

📧

Daily email overview

Overdue jobs, today's due & installs, open tasks, and low stock — in your inbox every morning via your own Gmail.

🗓️

CSV & calendar import

Sync jobs from spreadsheets or .ics calendar exports — re-imports update instead of duplicating.

🛟

Automatic backups

Full snapshots (database, files, and code) on startup, nightly, and before every import. One-click restore rolls everything back.

Quick start

Python 3.9+ is the only requirement. Your data lives in a local SQLite file — nothing leaves your machine.

git clone https://github.com/codybuckleywork-ops/signshop-dashboard.git
cd signshop-dashboard
pip install -r requirements.txt
python SignshopDashboard.py
# open http://localhost:5000 — done.
